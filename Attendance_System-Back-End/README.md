# Attendance System Back-End | الواجهة الخلفية لنظام الحضور

---

## English

This is the back-end part of the Face Gate Attendance System for Employees. It is built with Node.js and Express.js and provides the API and business logic for the system.

### How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. The server will run on [http://localhost:5000](http://localhost:5000) by default (or as configured).

### Features
- RESTful API for attendance, employees, departments, shifts, and users
- Face recognition integration
- Authentication and authorization

## العربية

هذا هو الجزء الخاص بالواجهة الخلفية لنظام بوابة الوجه - نظام حضور الموظفين. تم تطويره باستخدام Node.js وExpress.js ويوفر واجهات برمجية (API) ومنطق الأعمال للنظام.

### طريقة التشغيل
1. ثبّت الحزم:
   ```bash
   npm install
   ```
2. شغّل الخادم:
   ```bash
   npm start
   ```
3. الخادم يعمل افتراضياً على [http://localhost:5000](http://localhost:5000) (أو حسب الإعدادات).

### المميزات
- واجهات برمجية (API) لإدارة الحضور والموظفين والأقسام والورديات والمستخدمين
- تكامل مع نظام التعرف على الوجه
- نظام مصادقة وصلاحيات
